# Console Monopoly

The full game of Monopoly, playable from right in the terminal.

## Features
* Supports 2-4 players.
* Offers multiple board themes (e.g., Classic Monopoly, Minecraft).
* Fully interactive terminal UI with a dynamically drawn board and updating scoreboard.
* Includes core mechanics: buying properties, building houses, drawing Chance/Community Chest cards, and paying rent/taxes.
* Terminal-based interactive menus for managing your turn.

## How to Play
* Run `python main.py` in your terminal.
* Follow the on-screen prompts to add players and pick your theme.
* Use the menu to roll the dice, view your inventory, and manage your properties!
